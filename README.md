# Nginx, MySQL, PHP for Docker

## How to use

1. Build the image

```
docker-compose build;
```

2. Run the image

```
docker-compose up -d;
```

3. Compile the assets

```
npm run dev
```

4. Boot up a development server

```
php artisan serve
```

5. Now, you can point your browser to localhost, http://localhost:8000/

Thats it. The boilerplate is running.

Enjoy.
